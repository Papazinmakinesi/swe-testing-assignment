\# Quick-Calc — Software Testing Assignment



\## Project Description

Quick-Calc is a lightweight calculator application built for the Software Testing & Quality Assurance practical assignment.  

The application performs basic arithmetic operations and demonstrates a professional testing workflow using unit and integration tests.



Supported operations:

\- Addition

\- Subtraction

\- Multiplication

\- Division (with division-by-zero handling)

\- Clear/reset function
